SDA-DESPACHO / MANDATO ELECTRÓNICO

INTRODUCCIÓN

En el presente manual, el dueño, consignatario o consignante encontrará las pautas necesarias para poder interactuar dentro del aplicativo con mayor facilidad. El módulo SDA-Despacho "Mandato Electrónico" permite realizar las siguientes funcionalidades:

  • Registro de Autorización
  • Rectificación de Autorización
  • Anulación de Autorización

OBJETIVO

Guiar al dueño, consignatario o consignante en el uso de las opciones disponibles en el aplicativo SDA-Despacho "Mandato Electrónico".

ALCANCE

Este manual está dirigido al usuario de la opción portal del operador de comercio exterior. El dueño, consignatario o consignante para acceder a la opción "Mandato Electrónico." debería tener clave sol.
